Adding CSS to HTML Documents

14 Maret 2023

10 min read

CSS (Cascading Style Sheets) can be added to an HTML document in three ways:

1. Inline CSS

Inline CSS is added directly to an HTML element using the style attribute. This is the least recommended way of adding CSS to an HTML document as it can clutter the HTML code and make it difficult to maintain.

Example:

2. Internal CSS

Internal CSS is added to the head section of an HTML document using the <style> element. This method allows you to define CSS styles that will be applied to all elements on a single page.

Example:

3. External CSS

External CSS is stored in a separate file with a .css extension and linked to the HTML document using the <link> element in the head section. This method is the most recommended way of adding CSS to an HTML document as it separates the styling from the content and makes it easier to maintain.
