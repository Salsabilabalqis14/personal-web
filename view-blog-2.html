Declare a variable in JavaScript

20 Maret 2023

8 min read

In JavaScript, variables are used to store data values that can be used and manipulated throughout the code. To declare a variable in JavaScript, you need to use the keyword "var", "let", or "const" followed by the variable name.

Here are the different ways to declare a variable in JavaScript:

1. Using "var"

The "var" keyword is used to declare a variable in JavaScript. It is the oldest way to declare a variable, but it is still widely used.

2. Using "let"

The "let" keyword is a newer way to declare a variable in JavaScript. It is similar to "var", but it has some differences in terms of scope and hoisting.

3. Using "const"

The "const" keyword is used to declare a constant variable in JavaScript. Once a constant variable is declared, its value cannot be changed.

Overall, declaring a variable in JavaScript is relatively straightforward. By using the "var", "let", or "const" keyword followed by the variable name and an optional initial value, you can create variables that can be used and manipulated throughout your code.
